# Healthcare Dashboard

This is an interactive healthcare dashboard built using Dash and Plotly.

The dashboard provides insights into patient data such as age distribution, medical conditions, billing amounts, and admission trends.

---

## Features

- Patient demographics analysis (Age & Gender)
- Medical condition distribution
- Insurance provider comparison
- Billing amount filtering with slider
- Admission trends visualization (Line/Bar)

---

## Tech Stack

- Python
- Dash
- Plotly
- Pandas
- Bootstrap (dash-bootstrap-components)

---

## How to Run

1. Install required libraries:

pip install dash pandas plotly dash-bootstrap-components

2. Run the app:

python dashbaord.py

3. Open browser:

http://127.0.0.1:8050/

---

## Notes

- Dataset used: healthcare.csv
- Make sure all files are in the same directory
